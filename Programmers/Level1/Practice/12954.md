# 연습문제
## 1. x만큼 간격이 있는 n개의 숫자

![](https://images.velog.io/images/withcolinsong/post/e300ae28-ad65-44b5-8bde-664b2ac180b2/image.png)

### 1.1. 코드

```java
class Solution {
    public long[] solution(int x, int n) {
       long[] answer = {};
        
       answer = new long[n];
        
       for(int i=0; i<answer.length; i++) {
            answer[i] = x*(i+1);
        }
        return answer;
    }
}
```
