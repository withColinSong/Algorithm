## 1. 짝수와 홀수
![](https://images.velog.io/images/withcolinsong/post/fec09746-167b-4f75-b8e7-5bfb9baf2bc5/image.png)

링크 : https://programmers.co.kr/learn/courses/30/lessons/12937

### 1.1. 코드
```java
class Solution {
    public String solution(int num) {
        String answer = "";
        
        if(num % 2 == 0) {
            answer = "Even";
        }else {
            answer = "Odd";
        }
        return answer;
    }
}
```
